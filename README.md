# IDBI Wealth Avatar Dashboard 🚀

An interactive, gamified Wealth Management Portal designed to elevate financial literacy and portfolio tracking for retail investors. This full-stack system links a user's savings milestones directly to an evolving dynamic avatar profile.

---

## 🛠️ Tech Stack & Architecture

- **Backend:** Java 17, Spring Boot, Maven REST API Framework
- **Frontend:** HTML5, CSS3 Grid Layout, Modern JavaScript (Fetch API Architecture)
- **Data Exchange:** Synchronous JSON Payload Pipeline
- **Local Dev Server:** VS Code Live Server Environment

---

## 💡 Core Features Implemented

1. **Gamified Avatar Progression (Dynamic Core):** Automatically tracks and translates real-time financial metrics into interactive user milestones. (e.g., Upgrades portfolio ranks dynamically to **Growth Champion**).
2. **Behavioral Risk Badging:** Evaluates user risk profiles dynamically and adapts UI rendering configurations (e.g., **Aggressive Profile** badging).
3. **Smart Budget Track Simulator:** An automated client-side logic engine that alerts users instantly when a simulated expense crosses their threshold.
4. **Wealth Breakdown Analytics:** Interactive chart visualization panels mapping user asset distribution across investments, savings, and expenses.
